Paragraphs 段落樣式

段落樣式中的文字，盡量都不會去重新設定文字大小，所以規格都會依據其繼承，除了sub,sup有設定fontsize: smaller。

<p> 冤冤相報何時了，往事知多少。</p>

冤冤相報何時了，往事知多少。

冤冤相報何時了，<code>往事知多少</code>。
冤冤相報何時了，<strong>往事知多少</strong>。(strong or b)
冤冤相報何時了，<i>往事知多少</i>。(em or i)
冤冤相報何時了，<a> 往事知多少</a>。
冤冤相報何時了，<sub>往事知多少</sub>。
冤冤相報何時了，<sup>往事知多少</sup>。

冤冤相報何時了，往事知多少。
冤冤相報何時了，往事知多少。
冤冤相報何時了，往事知多少。
冤冤相報何時了， 往事知多少。
冤冤相報何時了，往事知多少。
冤冤相報何時了，往事知多少。

Table 表格

Table light

這是基本且輕巧型的表格，很適合在各種地方使用。

default 模式

<table class="table">
    <thead>
    <tr>
    	<th>#</th>
    	<th>First Name</th>
    	<th>Last Name</th>
    	<th>Username</th>
    </tr>
    </thead>
    <tbody>
    <tr>
    	<td>1</td>
    	<td>Mark</td>
    	<td>Otto</td>
    	<td>@mdo</td>
    </tr>
    ...
    </tbody>
</table>
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter

(4px,#ffffcc,center) 模式

# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter 
限制使用：table 標籤
+atablelight($padding,$Color,$textAlign)

aButton

這也是類似bootstrap按鈕，可以另外改顏色，預設為藍色，只要自己增加顏色就可以了。

這可以用在button、a.button, input[type="submit"]、input[type="button"]內。

<button type="submit">Login In</button>
<a class="button">Login In</a>

Login In

而他也可以結合bootstrap 2的icon

<a href="#" class="button"><i class="icon-ok icon-white"></i> Normal button can use bootstrap Icon,too.</a>

Normal button can use bootstrap Icon,too.

+aButton($color: $defaultColorGreen, $className: "success")
